内网其它不能直接访问外网的机器可以设置使用n147提供的代理服务。

全局的环境变量

在/etc/environment（不需要export），/etc/profile或/etc/profile.d/http_proxy.sh导出http_proxy和https_proxy

squid可以作为https代理，只要设置 https_proxy=http://2.2.2.147:3128， 即这个环境变量以http://开头。
